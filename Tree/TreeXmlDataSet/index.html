트리 구현을 위한 XML데이터의 계층 구조



XML의 노드는 반드시 시작 태그(start-tag)와 끝 태그(end-tag)가 쌍을 이루어야 합니다.



1

<rows>
  <row field=value>      <!-- 부모 노드 -->
    <row field=value />  <!-- 자식 노드 -->
    ...
    <row field=value />
  </row>
</rows>








부모 노드(위에서 rows tag)의 시작 태그와 끝 태그 사이에 들어 있는 또 다른 노드는 자식 노드(위에서 row tag)를 의미하게 됩니다.



이 데모의 데이터에서 주목할 부분은 row라는 이름의 노드(node 또는 엘리먼트 element)입니다.
이 노드는 트리의 계층구조를 표현하기 위한 엘리먼트(element) 입니다.

// 현재 노드의 아이콘을 9 또는 10으로 변경하기
var current = treeView.getCurrent();
if (current && current.dataRow) {
  var iidx = treeDataProvider.getIconIndex(current.dataRow);
  if (iidx == 9) {
    treeDataProvider.setIconIndex(current.dataRow, 10);
  } else {
    treeDataProvider.setIconIndex(current.dataRow, 9);
  }
